# Autolitics Studio
## Vehicle Test Drive Scorecard

Purpose:  
Create a clean, printable scorecard that buyers can take with them during dealership visits and test drives to objectively evaluate and compare vehicles.

The scorecard should align with the **Autolitics Evaluation Model** and help buyers capture observations while experiencing vehicles in the real world.

Format:  
• Printable PDF  
• Single page preferred (two pages maximum)  
• Clean layout with space for notes  
• Professional, minimal design  

This resource should feel like a **decision tool**, not a checklist.

# Design Notes

The scorecard should feel:

• clean
• structured
• analytical
• easy to complete during a test drive
- should be a decision tool, not a checklist
- should be easy to fill out during a test drive
- should be easy to read and understand
- should be easy to print and take with you to the dealership
- ensure pagination is handled correctly

Avoid visual clutter.

Use subtle section dividers and clear scoring fields.

Brand styling should align with Autolitics Studio design language.
